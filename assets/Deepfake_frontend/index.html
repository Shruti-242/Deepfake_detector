Deepfake Detector

This is a simple deepfake detector that allows you to upload an image or video to analyze whether it might be a manipulated or synthetic media (a "deepfake").

Deepfake detection algorithms analyze various features within the media to identify inconsistencies or artificial patterns that are often introduced during the deepfake creation process.

To use this detector, simply click the "Choose File" button, select the image or video file you want to analyze, and the system will process it and display the prediction and its confidence level below.
